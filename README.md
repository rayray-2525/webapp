# 扇風琴 型紙設計

中心から外側へ「ド・レ・ミ・ファ・ソ・ラ・シ・ド」の順に鳴る扇風琴の型紙を設計する静的Webアプリです。

## 使い方

`index.html` をブラウザで開いてください。

- 音階、調律モデル、用紙、扇の角度、半径、音の間隔、羽の回転数、音穴の幅を調整できます。音穴の長さは、穴と穴の間の壁の長さが1:1になるように自動計算されます。
- SVGの型紙をダウンロードできます。
- ブラウザの印刷機能で実寸印刷できます。
- 試聴ボタンまたは音の行を押すと、目標音を確認できます。

紙や厚みで実音は変わるため、印刷後は舌片を少しずつ短くしながら調整してください。
